# 第01章　スケーラブルな言語

## rf

### アクターライブラリ

ここで解説されているアクターライブラリ (scala.actor 系) は
Scala version 2.10 で deprecated とされ、2.11 で除去されている。

2.10 からは [Akka](http://akka.io/) ライブラリが
Scala 標準のアクターライブラリとなっている。
